## ğŸ¨ Usage

### Adding a Task

1. Fill in the task form on the left side:
   - **Subject**: Enter the subject or course name
   - **Task**: Describe what needs to be done
   - **Priority**: Select High, Medium, or Low
   - **Due Date**: Optionally set a due date
2. Click "Add Task" to save

### Managing Tasks

- **Complete Task**: Click the "Complete" button to mark a task as done
- **Undo Completion**: Click "Undo" to revert a completed task
- **Delete Task**: Click "Delete" to remove a task permanently
- **Search**: Use the search bar to filter tasks by keyword
- **Filter Tabs**: Switch between "All Tasks", "Pending", and "Completed" views

### Statistics

View your progress in the Statistics card:
- Total Tasks: All tasks you've created
- Completed: Number of finished tasks
- Pending: Number of tasks still to do

### Clear All Tasks

Click the "Clear All" button in the navbar to remove all tasks (with confirmation).
